# Documentation Generation Modules

This directory contains modular components for comprehensive documentation generation using real tools and frameworks.

## Module Structure

### Core Modules

- api-documentation.md - OpenAPI/Swagger generation patterns for FastAPI, Express, and other frameworks
- code-documentation.md - Source code documentation with Sphinx, TypeDoc, JSDoc
- user-guides.md - User guide and tutorial creation patterns
- multi-format-output.md - HTML, PDF, and static site generation with real tools

### Tool Categories

API Documentation Tools:
- FastAPI automatic OpenAPI generation
- swagger-jsdoc for Express/Node.js
- Redoc and Swagger UI for rendering

Code Documentation Tools:
- Sphinx with autodoc for Python
- TypeDoc for TypeScript
- JSDoc for JavaScript
- mkdocstrings for MkDocs integration

Static Site Generators:
- Nextra (Next.js-based)
- Docusaurus (React-based)
- VitePress (Vue-based)
- MkDocs with Material theme

### Usage Patterns

1. API Documentation: Reference api-documentation.md for OpenAPI patterns
2. Code Analysis: Reference code-documentation.md for docstring-based docs
3. User Guides: Reference user-guides.md for tutorial creation
4. Output Formats: Reference multi-format-output.md for publishing
